# V-XN

> A tiny astromech companion for Raspberry Pi.

V-XN is an OLED status display inspired by the personality of
Star Wars astromechs.

It quietly monitors Home Assistant, displays system status,
judges the operator's life choices, and occasionally reminds
everyone that it's probably DNS.

## Planned Features

- SSD1306 OLED support
- Randomized status screens
- Boot animation
- MQTT integration
- Home Assistant integration
- Aurebesh font support
- Chopper-inspired personality mode
